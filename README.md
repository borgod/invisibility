Invisibility program
====================


1. Instructions : Download the following files in a common folder in your local
   machine, for example /temp/
   -. invisibility_bg.ipynb
   -. background.png
   -. target.mov

2. Run the script invisibiluty_bg.ipynb in, for example, Visual Studio Code

4. The script will produce a new file, called output.mov, with the invisibility effect
   on my jacket (I didnt have a cloak available)

5. You can skip all of the above, and get the result directly by downloading
   the file in /results/output.mov, showing the same that you would get with
   steps (1), (2), and (3)
